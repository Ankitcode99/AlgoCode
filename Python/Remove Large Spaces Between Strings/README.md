<h1> Removing Large Spaces Between Strings</h1>
<h2>Description</h2>

Given a string of words with lots of spaces between the words , remove all the unnecessary spaces.

<br>
<h2>Example</h2>

Input: I       Love     Indian    Army

Output: I Love Indian Army
